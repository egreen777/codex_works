# codex_works

codex를 사용한 연습용 Project 모음

## 구성

- `todo.html`
  - 정적 To-do HTML 파일 (연습용)

- `invest_backtest.md`
  - 백테스팅 프로젝트의 상위 작업 가이드

- `run_backtest.sh`
  - 백테스터 실행 예시 명령 모음

- `backtester/`
  - Python 기반 ETF 백테스터
  - Yahoo Finance 일봉 데이터를 다운로드해 `backtester/data/`에 CSV로 캐시
  - 전략별 성과표와 로그를 출력

- `tradingview/`
  - Python 백테스터 전략 중 TradingView에서 구현 가능한 전략의 Pine Script
  - 차트 위에서 전략 신호와 판단 근거를 시각화하는 용도

- `sapporogolf/`
  - Sapporo Golf 여행지도를 만들기 위한 Google Map KMZ 파일
